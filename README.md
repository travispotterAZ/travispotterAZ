## Featured Projects

**[Programming Contest Management System](https://github.com/travispotterAZ/SFWE405_Software_Architecture_and_Design)**  
A Spring Boot REST API with JPA persistence, eligibility-enforced team registration, and asynchronous person intake via RabbitMQ. Built incrementally across four assignments covering domain modeling, persistence, REST, and messaging.

**[Pipelined MIPS Processor](https://github.com/travispotterAZ/ECE_369a_Fundamentals_of_Computer_Organization)**  
A fully pipelined 32-bit MIPS processor in Verilog, deployed on a Nexys4 DDR FPGA. Implements a 5-stage pipeline with hazard detection, branch resolution, and support for 20+ MIPS instructions.

**[Hybrid Library Record Store](https://github.com/travispotterAZ/HybridLibraryRecordStore---SQL)**  
A SQLite-backed vinyl record library with user authentication, loan and purchase workflows, and database performance analysis. Record data sourced from the Spotify API.
